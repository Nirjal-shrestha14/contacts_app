# 📱 Contacts App

A Flutter application for managing contacts using **sqflite** as the local database.

---

## ✨ Features

- View all saved contacts
- Add new contacts
- Edit existing contact details
- Mark/unmark favorite contacts
- Delete contacts
- Uses `sqflite` for local storage
